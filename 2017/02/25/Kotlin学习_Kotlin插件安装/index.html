Kotlin学习_Kotlin插件安装

什么是Kotlin？

Kotlin是针对JVM、Android 和浏览器的静态编程语言！
100% 与 Java™ 可互操作！

Kotlin在Android上的使用

Kotlin可以直接将布局的id来当成变量使用：

Java在写Bean类的时候是这样写的

而在Kotlin中直接用数据类(Data Classes)写

是不是很方便，很简洁，下面就介绍如何安装Kotlin插件

Kotlin的插件安装

Settings->Plugins->Browse Repositories->搜索Kotlin

项目添加Kotlin（Android Studio）

已存在的项目直接转换为Kotlin

Code->Convert Java file to Kotlin file

  1. 转换完成后，会要求配置Kotlin
  2. 选择Modules和Kotlin的版本
  3. Sync Gradle
  4. 转换后的文件，会有些语法之类的错误，这个后面再说

新建项目

  1. Project的gradle

    • 在buildscript里面添加kotlin的版本，当前是1.0.4

    • 在dependencies里面添加Kotlin依赖

  2. Module的gradle

    • 添加apply plugin

    • 在dependencies添加依赖

  3. 运行app
